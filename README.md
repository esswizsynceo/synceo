This site allow users to download the program that will run 
on the desk computer to run the server Synceo.

Synceo give you the ability to easily share files between your 
smartphone and your desk computer.

The smartphone application is available on PlayStore® platform,
so users of this application are allowed to use this website to 
get the server counterpart of their file sharing app.
